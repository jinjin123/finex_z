1、发送验证码接口

说明：

接口负责人：刘富国
请求方式：POST
请求示例： 
{
    "server": "SendCode",
    "ver": "1.0.0",
    "token": "xxxxxx",
    "data": {
        "username": "lfg",  //用户名
        "phone_imei": "xxxx"  //手机唯一标示imei码
    }
}
成功返回： 
{
    "error":"0",
    "msg":"success",
    "data":{
        "is_send":"1" // 发送成功
    }
}
失败返回：（有多种情况） 
{
    "data": {},
    "error": "错误码",
    "msg": "错误说明"
}

接口负责人：刘富国
请求方式：POST
请求示例： 
{
    "server": "SignIn",
    "ver": "1.0.0",
    "data": {
        "username": "xxxxx", // 用户名(rsa+base64加密)
        "password": "xxxx", // 登录密码(rsa+base64加密)
        "phone_imei": "2DB86836-6CB1-462B-83B5-D73EDF89F9D8" //手机唯一标示imei码

    }
}
需要注册时的返回（短信登录，没找到手机号码，需要用户注册）： 
{
  "data": {
    "need_register": "1" // 需要注册
  },
  "error": "0",
  "msg": "success"
}
登录成功返回： 
{
  "data": {
    "username": "lfg", // 用户名
    "is_bind": "1" // 0：未绑定手机；1：已绑定手机
    "phone": "186****7953", //用戶手機
    "token": "xxxxxxxxxxxxx" // TOKEN
  },
  "error": "0",
  "msg": "success"
}
失败返回：（有多种情况） 
{
    "data": {},
    "error": "错误码",
