1、手机口令获取接口

说明：

接口负责人：刘富国
请求方式：POST
请求示例： 
{
    "server":"CreateWatchwordToken", // 请求Server
    "ver":"1.0.0", // 版本号
    "token": "xxxxxx", // TOKEN
    "data":{}
}
成功示例： 
{
    "data": {
        "current_time": "1508480966",  //時間戳
        "secret_key": "7504_wXU2IFbYN5"  //用戶秘鈅
    },
    "error": "0",
    "msg": "success"
}
失败返回：（有多种情况） 
{
    "data": {},
    "error": "错误码",
    "msg": "错误说明"
}

2、绑定手机接口

说明：绑定手机接口

接口负责人：刘富国
请求方式：POST
请求示例： 
{
    "server":"BindAccount", // 请求Server
    "ver":"1.0.0", // 版本号
    "token": "xxxxxx", // TOKEN
    "data": {
        "code": "4346",          //短信验证码
        "phone_imei": "2DB86836-6CB1-462B-83B5-D73EDF89F9D8"  //手机唯一标示imei码
    }
}
成功示例： 
{
    "data": {
        "is_bind": "1"  //已绑定
    },
    "error": "0",
    "msg": "success"
}
失败返回：（有多种情况） 
{
    "data": {},
    "error": "错误码",
    "msg": "错误说明"
}

3、校验手机綁定接口

说明：校验手机綁定接口

接口负责人：刘富国
请求方式：POST
请求示例： 
{
    "server":"CheckUserBind", // 请求Server
    "ver":"1.0.0", // 版本号
    "token": "xxxxxx", // TOKEN
    "data": {}
}
成功示例： 
{
    "data": {
        "is_bind": "1"  // 0：未绑定手机；1：已绑定手机
    },
    "error": "0",
    "msg": "success"
}
失败返回：（有多种情况） 
{
    "data": {},
    "error": "错误码",
    "msg": "错误说明"
}

4、语言版本切换

说明：语言版本切换

接口负责人：刘富国
请求方式：POST
请求示例： 
{
    "server": "SetLang",
    "ver": "1.0.0",
    "token": "xxxxxx", // TOKEN  （可选）
    "data": {
        "var_language": "zh-tw",  // 允许切换的语言列表 ：英文：en-us,简体 zh-cn,繁体 zh-tw
        "phone_imei": "xxxx"  //手机唯一标示imei码
    }
}
成功示例： 
{
    "data": {
        "is_success": "1"
    },
    "error": "0",
    "msg": "success"
}
失败返回：（有多种情况） 
{
    "data": {},
    "error": "错误码",
    "msg": "错误说明"
}
