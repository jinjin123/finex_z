1、查询用户自己登录操作日志接口

说明：只显示前30天的操作日志

接口负责人：刘富国
请求方式：POST
请求示例： 
{
    "token": "xxxx",
    "server": "GetUserLogByUid",
    "ver": "1.0.0",
    "data": {
        "page": "1",  //页数
        "limit": "10"  // 每页显示行数
    }
}
成功返回： 
{
    "data": {
        "total": "30",  //总数
        "list": [
            {
                "id": "42",
                "type_name": "登录",  //操作类型
                "ip_area": "本机地址",  // IP所属区域
                "add_time": "1509006098"  //时间戳
            }
        ],
        "pager": {
            "current_page": "1",  //当前页数
            "last_page": "",
            "next_page": "2",
            "total_pages": "30"
        }
    },
    "error": "0",
    "msg": "success"
}
    或者：
    {
    "data": {
        "total": "30",
        "list": {},
        "pager": {
            "current_page": "100",
            "last_page": "99",
            "next_page": "",
            "total_pages": "15"
        }
    },
    "error": "0",
    "msg": "success"
}
失败返回：（有多种情况） 
{
    "data": {},
    "error": "错误码",
    "msg": "错误说明"
}
