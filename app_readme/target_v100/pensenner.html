3.财务总账接口

说明：

接口负责人：张锡文
请求方式：POST
请求示例： 
					{
    "token": "xxxx",
    "server": "GetUserFinance",
    "ver": "1.0.0",
    "data": {
        "currency_id":1, // 1,2,3,4,5 对应不同币种  (可选)
		"finance_type":1,//1-18分别对应不同状态  （可选）
		"page":1,
		"limit":10,
    }
}
成功返回： 
{
    "data": {
        "total": "2",
        "list": [
            {
                "content": "系统充值", //内容
                "type": "1",//收入=1/支出=2
                "money": "10000.0000",//价格
                "add_time": "1510198149"//添加时间
            },
        ],
        "pager": {
            "current_page": "1",
            "last_page": "",
            "next_page": "",
            "total_pages": "1"
        }
    },
    "error": "0",
    "msg": "success"
}
失败返回：（有多种情况） 
{
    "data": {},
    "error": "错误码",
    "msg": "错误说明"
}
