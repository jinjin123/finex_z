1、强制更新

说明：

    口令APP: nat1.target.com/App/Update/index

    target APP: nat1.target.com/AppTarget/Update/index

接口负责人：刘富国							
请求方式：POST
请求参数：Update/index

POST到服务器的json数据：
{
	"app_platform": "app_watchword", //平台:app_watchword口令APP，app_target系统APP
	"ver": "1.0.0",					// 版本号
	"os_ver": "android9",			// 操作系统版本
	"os": "android", 				//os操作系统：值:android | ios
	"hash": "75MWHZrpRhnbLDRzM6m84aPQ9HxYvqpa",  // 加密hash
	"token": ""  					// app端保存的token，可以为空
	"phone_imei":"xxxxx"      		//设备imei ，可以为空
}
成功返回：
							{
    "data": {
        "is_need_update": "1",  // 0-最新版本，1-有新版本更新，2-需强制更新
        "update_link": "",
        "call_ver": "1.0.1",  //当前版本
        "last_ver": "2.0.0",  //最新版本
        "update_tip": "手机绑定", // 更新提示
        "token": "xxxxx"
    },
    "error": "0",
    "msg": "success"
}
