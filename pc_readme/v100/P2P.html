1.买入

说明：买入接口

接口负责人：张锡文
请求方式：POST
接口地址：http://nat1.target.com/OffTrading/buying
请求示例： 
请求参数 
{    
        "orderId":xxxxx      //订单ID
	"bankId":1                     //银行ID
	"currencyId":1        //币种ID
	"tradePwd":xxxxxx          //交易密码
}


{
 code: 200,
 msg: "买入成功"
 }

{
 code: 201,
 msg: "xxxxxx"
 }


补充: 状态码code 只有200的时候为成功
其他状态一律是失败  并且会返回对应的错误信息

2.卖出

说明：卖出接口

接口负责人：李江
请求方式：POST
接口地址：nat1.target.com/OffTrading/selling
请求示例： 
请求参数
{
    "userArea":1,           //地区id
	"currencyId":1   ,      //卖出币种id
	"coinPrice":"1234",     //卖出单价
	"coinNum":12,           //卖出数量
	"userBankId":1,         //银行卡id
	"transpwd":"132456",    //交易密码
	"verifyCode":  1,       //图片验证码
}


{
 code: 200,
 msg: "卖出成功",
}
失败返回：（有多种情况） 
{
	"code": 201,
	"msg": "xxxx",
	"data": []
}


补充: 状态码code 只有200的时候为成功
其他状态一律是失败  并且会返回对应的错误信息
