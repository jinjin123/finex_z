2、用户下单

说明：用户挂单

    接口负责人：李润清
    请求方式：POST
    接口地址：nat1.target.com/CurrencyTrading/getUserMainOrderList
    请求示例： 
                            {
                            "tradeType" : "1", // 交易区,1表示btc交易区，2表示VP交易区
                            "priceType" : "1", // 价格类型，1表示限价单，2表示市价单
                            "transactionType" : "1", // 交易类型，1表示买入，2表示卖出
                            "entrustType" : "1",  // 委托类别;例如：1表示ltc/btc 2表示etc/btc 3表示eth/btc 4表示bcc/btc
                            
                            "entrustPrice" : "0.01632682", // 单价
                            "leaveNum" : "1", // 数量
                            "tradePwd" : "123456" // 资金密码
                        }
    成功： 
                            {
                            "code": 200,
                            "msg": "",
                            "data": []
                        }
    失败返回： 
                            {
                            "code": 201,
                            "msg": "xxxx",
                            "data": []
                        }
